# Travaux Pratiques – Programmation Python  
Université Quisqueya – Génie Informatique  

---

## Description

Ce repository contient l'ensemble des travaux pratiques (TP) du cours de Programmation Python.

Chaque séance possède son propre dossier contenant :

- Un énoncé (`enonce.md`)
- Les fichiers d'exercices (`.py`)
- Éventuellement des ressources complémentaires

L'objectif est de fournir un espace centralisé et structuré pour les activités pratiques du cours.

---

## Objectifs pédagogiques

À travers ces TP, les étudiants développeront leur capacité à :

- Comprendre et manipuler les variables
- Utiliser les types fondamentaux (`int`, `float`, `str`, `bool`)
- Mettre en œuvre les structures conditionnelles
- Utiliser les boucles
- Décomposer un problème en fonctions
- Structurer un programme Python correctement
- Adopter de bonnes pratiques de codage

---

## Règles générales

- Chaque exercice doit être réalisé dans un fichier séparé.
- Les noms des fichiers doivent être respectés.
- Le code doit être correctement indenté.
- Les variables doivent avoir des noms explicites.
- Les affichages doivent être clairs et compréhensibles.

---

## Pré-requis

- Python 3.x installé
- Un éditeur de code (VS Code, PyCharm, etc.)
- Savoir exécuter un fichier Python depuis le terminal

Pour vérifier l'installation :

```bash
python --version
```

---

## Comment utiliser ce repository

1. Cloner le repository :

```bash
git clone <lien_du_repository>
```

2. Accéder au dossier :

```bash
cd python-tp-uniq
```

3. Ouvrir le dossier dans votre éditeur de code.

4. Se rendre dans le dossier correspondant à la séance.

5. Lire le fichier `enonce.md` avant de commencer.

---

## Méthodologie recommandée

Avant de coder :

* Lire attentivement l'énoncé
* Identifier les entrées
* Identifier les traitements
* Identifier les sorties
* Écrire une logique simple sur papier si nécessaire

---

## Bonnes pratiques encouragées

* Utiliser des noms de variables explicites
* Ajouter des commentaires lorsque nécessaire
* Tester son programme avec plusieurs cas
* Corriger les erreurs d'indentation
* Éviter le copier-coller sans compréhension
